# TOP V10 React (base inicial)

## Rodar local

```bash
npm install
npm run dev
```

## Estrutura

- `src/data`: catalogo de procedimentos
- `src/domain`: regras de preco e utilitarios
- `src/storage`: persistencia local
- `src/hooks`: estado de planos do vendedor
- `src/components`: blocos de interface
- `src/screens`: composicao de telas

## Escopo desta fase

- Mundo do vendedor funcional em React
- Regras de minimo/campanha/edicao manual por item
- Persistencia basica da sessao em localStorage
